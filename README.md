# Fastify Jumpstart

A starter kit template for Fastify Typescript REST APIs.

## Features
- Typescript
- OpenAPI v3 with fastify-swagger
- Testing with Tap
- ESLint
- Database Agnostic

## Available Optional Plugins
- Prometheus for monitoring
